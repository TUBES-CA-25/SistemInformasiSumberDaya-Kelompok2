# Sistem Management Sumber Daya

Aplikasi PHP Native untuk manajemen sumber daya

## Struktur Folder

```
SistemManagementSumberDaya/
├── app/
│   ├── config/          # Configuration files
│   ├── controllers/     # Controller classes
│   ├── models/          # Model classes
│   ├── views/           # View files (HTML)
│   ├── helpers/         # Helper functions
│   ├── middleware/      # Middleware classes
│   └── routes/          # Route definitions
├── public/
│   ├── index.php        # Entry point
│   ├── css/             # CSS files
│   ├── js/              # JavaScript files
│   └── images/          # Image files
├── storage/
│   ├── logs/            # Log files
│   └── uploads/         # Uploaded files
└── README.md
```

## Instalasi

1. Clone atau download project
2. Pastikan PHP dan MySQL sudah terinstall
3. Konfigurasi database di `app/config/config.php`
4. Buka browser dan akses `http://localhost/SistemManagementSumberDaya`

## Persyaratan

- PHP 7.0 atau lebih tinggi
- MySQL 5.7 atau lebih tinggi
- XAMPP atau server lokal lainnya

## Penggunaan

Ikuti dokumentasi di folder aplikasi untuk membuat controller, model, dan view.
